# rocky9Tpa

This repo allows you to run  a tpa container to orchestrate additional tpa deploys on your existing while keeping it local to your laptop or desktop.

There are several way to run this based on your architecture and OS. Having said that,  click on the links below for the appropriate set of instruction

[Running on Windows](https://github.com/jtorral/rocky9Tpa/blob/main/winDoc.md)

[Running on Linux](https://github.com/jtorral/rocky9Tpa/blob/main/linuxDoc.md)

[Running on ARM](https://github.com/jtorral/rocky9Tpa/blob/main/macDoc.md)
